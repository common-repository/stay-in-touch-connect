Stay In Touch Plugin For Adding subscribers To Stayintouch Panel Lists.
 
== Description ==
 
This Plugin allows you to connect to Stay In Touch newsletter system via APIs. This Plugin works on API service of Stay In Touch.

Stay In Touch (SIT) email offers an integrated, automated and robust feature set built to help the enterprise of any size. A powerfully simple and simply powerful email marketing solution designed to help you create and send compelling campaigns with ease. It is used for Email Automation, Newsletter and Marketing.

Features

    1. Inbuilt Template Builder
    2. Automate Your campaigns
    3. Segment Your Contacts
    4. Get Statistical Reports
    5. Track Opens/Clicks 
    6. Track 24 Hours Performance


== 3rd Party or external service ==

This Plugin Allows you to connect to stayintouch application https://www.stayintouch.co.in/ via API to its App (http://in02.stayintouch.co.in/api/v1/) .

The External APIs Are used for the following.

    1. Fetch User Token (GET Methods)
    2. Fetch Lists Available for Loged In User (GET Methods)
    3. Subscribe a Subscriber to the List (POST Methods)

 
== Installation ==
 
This section describes how to install the plugin and get it working.
 
    Step 1 - Upload stayintouch.zip to the /wp-content/plugins/ director

    Step 2 - Extract stayintouch.zip to the /wp-content/plugins/ directory

    Step 3 - Activate the plugin through the ‘Plugins’ menu in WordPress

    Step 4 - Goto Stay In Touch connection page and login to the system

    Step 5 - Use Stay In Touch shortcode on the page
